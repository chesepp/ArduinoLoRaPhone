# ArduinoLoRaPhone
A pocket Lora Terminal that runs off of an arduino

Materials Required:
Arduino Uno (Im using an R4 with Wifi)
Reyax RYLR 998 UART LoRa transmitter
Rotary Encoder
bunch of buttons
I2C oled screen

everything is wired separately so it shouldnt be that hard to fashion a circuit out of these parts that will work, Buttons are connected to digital pins, REYAX RYLR 998 IS CONNECTED TO 3V3 NOT 5V
